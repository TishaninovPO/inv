Free Barcode Component

For Delphi 1, 2, 3 and 4

Features

Print barcodes at any angle. (0 to 359 degree)

Width property to precalculate the width of a barcode

Properties

Height

Height of Barcode in Pixels

Text

Contents of the barcode

Top

Y-Position of the upper left corner of the barcode

Left

X-Position of the upper left corner of the barcode

Modul

Width of the smallest line in the barcode

Ratio

Ratio between a wide and a narrow line in a barcode. Normal Values are from 2.0 to 3.0

Typ

Defines the Type of the barcode. Possible values are

bcCode_2_5_interleaved,

bcCode_2_5_industrial,

bcCode_2_5_matrix,

bcCode39,

bcCode39Extended,

bcCode128A,

bcCode128B,

bcCode128C,

bcCode93,

bcCode93Extended,

bcCodeMSI,

bcCodePostNet,

bcCodeCodabar

bcCodeEAN8

bcCodeEAN13

bcCodeUPC_A

bcCodeUPC_E0

bcCodeUPC_E1

bcCodeUPC_Supp2

bcCodeUPC_Supp5

bcCodeEAN128A

bcCodeEAN128B

bcCodeEAN128C

Checksum

Enable or disable checksum calculation for Code39, EAN8 or EAN13

CheckSumMethod

csmNone
csmModulo10 (default for EAN 8/13)

Angle

Angle the the barcode. (use 0 to 359 degree)

Showtext

Enable or disable the printing in plain text. (for testing)

bcoNone
bcoCode
bcoTyp
bcoBoth
see Demo programm for visual effect of this options

Width

Width of barcode (read only)

Color

Color of the Background (default is white)

ColorBar

Color of the bars (default is black)

 

Procedures

  • procedure DrawBarcode(Canvas:TCanvas);
    this procedure draws the barcode on the specified canvas.
  • procedure DrawText(Canvas:TCanvas);
    see property     Showtext.

 

Example

 

Barcode1.Top := 200;
Barcode1.Left := 230;
Barcode1.Angle := 70;
Barcode1.Typ := bcCode_2_5_interleaved;
Barcode1.DrawBarcode(Image1.Canvas);
